11. අන්තර්ජාලය සහ විද්‍යුත් තැපෑල

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී අන්තර්ජාලය, එහි ක්‍රියාකාරිත්වය සහ විද්‍යුත් තැපෑල භාවිතය පිළිබඳව මූලික දැනුම ලබා ගනිමු.

හැඳින්වීම

අන්තර්ජාලය (Internet) යනු ලොව පුරා විසිරී ඇති පරිගණක ජාල මිලියන ගණනක් එකිනෙකට සම්බන්ධ කරන විශාලතම ජාලයයි. මෙමඟින් තොරතුරු සහ සේවා හුවමාරු කර ගැනීමට හැකියාව ලැබේ.

1. අන්තර්ජාලය ක්‍රියාත්මක වීම

අන්තර්ජාලය පදනම් වී ඇත්තේ සේවාලාභී-සේවාදායක (Client-Server) ආකෘතිය මතය.

[Image of Client Server architecture for the web]

1.1. සේවාදායක (Servers) සහ සේවාලාභී (Clients)

  • සේවාදායක (Server): වෙබ් අඩවි, දත්ත සහ අනෙකුත් ලිපිගොනු ගබඩා කර ඇති, අන්තර්ජාලයට නිරන්තරයෙන් සම්බන්ධ වී ඇති බලවත් පරිගණක වේ.
  • සේවාලාභී (Client): ඔබගේ පුද්ගලික පරිගණකය, දුරකථනය හෝ ටැබ්ලටය වේ. මේවා බ්‍රවුසරයක් මඟින් සේවාදායක වෙතට ඉල්ලීම් (Requests) යවා තොරතුරු ලබා ගනී.

1.2. ප්‍රොටෝකෝල (Protocols)

දත්ත හුවමාරු කර ගැනීමට භාවිතා කරන නීති මාලාවන්ය.

  • TCP/IP: අන්තර්ජාලය හරහා දත්ත යවන සහ ලබන ආකාරය තීරණය කරන මූලිකම ප්‍රොටෝකෝලය.
  • HTTP (HyperText Transfer Protocol): වෙබ් පිටු හුවමාරු කර ගැනීමට භාවිතා කරයි.
  • HTTPS (HTTP Secure): ආරක්ෂිත දත්ත හුවමාරුව (Secure transactions) සඳහා භාවිතා කරයි.

1.3. IP ලිපිනය සහ Domain නාම

  • IP ලිපිනය (IP Address): අන්තර්ජාලයට සම්බන්ධ සෑම උපාංගයක්ම හඳුනා ගන්නා සංඛ්‍යාත්මක ලිපිනයකි (උදා: 192.168.1.1).
  • Domain නාම (Domain Name): වෙබ් අඩවි මතක තබා ගැනීමට පහසු වන නාමයන්ය (උදා: google.com).
  • DNS (Domain Name System): Domain නාමයක් අදාළ IP ලිපිනයට පරිවර්තනය කරන පද්ධතියයි.

2. වෙබ් බ්‍රවුසර් සහ සෙවුම් යන්ත්‍ර

2.1. වෙබ් බ්‍රවුසර් (Web Browsers)

වෙබ් බ්‍රවුසරයක් යනු සේවාදායකයෙන් ලබා ගන්නා වෙබ් පිටු පරිශීලකයාට බැලීමට හැකි ආකාරයට විදැහුම් කරන (Render) මෘදුකාංගයකි.

  • ජනප්‍රිය උදාහරණ: Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari.

2.2. සෙවුම් යන්ත්‍ර (Search Engines)

සෙවුම් යන්ත්‍ර යනු අන්තර්ජාලය පුරා ඇති තොරතුරු සොයා ගැනීමට පරිශීලකයින්ට උපකාර කරන වෙබ් අඩවි වේ.

  • ජනප්‍රිය උදාහරණ: Google, Bing, Yahoo.
  • ක්‍රියාත්මක වන ආකාරය: Crawl (රිංගා යාම), Index (අංකනය) සහ Search (සෙවීම) යන පියවර අනුගමනය කරයි.

3. විද්‍යුත් තැපෑල (Email)

විද්‍යුත් තැපෑල යනු ජාලයක් හරහා ඩිජිටල් පණිවිඩ හුවමාරු කර ගැනීමේ ක්‍රමයයි.

3.1. විද්‍යුත් තැපැල් ලිපිනයක ව්‍යුහය

පරිශීලක නාමය @ Domain නාමය

උදා: studentname@gmail.com

3.2. තැපැල් ශිෂ්ටාචාරය (Email Etiquette)

හොඳ පුරුදු වැරදි පුරුදු
විෂය ක්ෂේත්‍රය (Subject) පැහැදිලිව සඳහන් කිරීම. ලිපියේ අකුරු සියල්ල ලොකු අකුරින් (CAPITAL LETTERS) ලිවීම.
විධිමත් ආචාර විධි (Formal Greeting) භාවිතා කිරීම. විශාල ගොනු (Attachments) අනතුරු ඇඟවීමකින් තොරව යැවීම.

වැඩිදුර අධ්‍යාපනය සඳහා

අන්තර්ජාලය සහ විද්‍යුත් තැපෑල පිළිබඳ මෙම වීඩියෝව නරඹන්න.

වීඩියෝව නරඹන්න

පුහුණු ප්‍රශ්න

  1. IP ලිපිනයක් සහ Domain නාමයක් අතර වෙනස කුමක්ද?
  2. DNS පද්ධතියේ කාර්යය කුමක්ද?
  3. වෙබ් බ්‍රවුසරයක ප්‍රධාන කාර්යයන් දෙකක් ලියන්න.
  4. විද්‍යුත් තැපැල් ශිෂ්ටාචාරයේ (Email Etiquette) වැදගත්කමක් ලියන්න.
නිබන්ධන මෙනුවට ආපසු
