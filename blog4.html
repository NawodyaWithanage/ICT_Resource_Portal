Microsoft Access

ලේඛක: Nawodya Withanage | December 17, 2025   15 මිනිත්තු

O/L ICT practical exam එකේ ඉතාම වැදගත් කොටසක් තමයි දත්ත සමුදායන් (Databases) සහ Microsoft Access භාවිතය. Database යනු organized data collection එකක් – ඒකෙන් data store කරලා efficiently manage කරන්න පුළුවන්. Microsoft Access යනු simple database create කරන්න හොඳ tool එකක්.

Database යනු කුමක්ද?

Database එකක main components: Tables, Queries, Forms, Reports.

Simple Example: Student Database

අපි simple student database එකක් create කරමු. Fields: StudentID, Name, Grade, Subject.

Step 1: Table Create කිරීම

  1. Access open කරලා Blank Database create කරන්න.
  2. Create > Table Design.
  3. Fields add කරන්න (StudentID - AutoNumber, Name - Short Text, Grade - Number).
  4. Save as "Students".

Step 2: Query Create කිරීම

Query යනු data filter කරන්න හෝ search කරන්න භාවිතා වෙනවා.

  1. Create > Query Design.
  2. Table add කරලා fields drag කරන්න.
  3. Criteria දාන්න (e.g., Grade > 75).

Step 3: Form Create කිරීම

Form යනු data entry සඳහා easy interface එකක්.

Step 4: Report Create කිරීම

Report යනු data print කරන්න හෝ summarize කරන්න.
