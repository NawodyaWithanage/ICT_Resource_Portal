මූලික Programming Concepts සහ Flowcharts

Nawodya Withanage | December 17, 2025   15 මිනිත්තු

O/L ICT විෂයේ programming section එක ඉතාම වැදගත්. මෙතනදී Algorithm, Pseudocode, Flowcharts, Variables, Conditions සහ Loops ගැන අපි ඉගෙන ගන්නවා.

Algorithm යනු කුමක්ද?

Algorithm යනු problem එකක් solve කරන step-by-step instructions එකතුවක්. එය සරලව pseudocode හෝ flowchart විදිහට පෙන්විය හැකියි.

Flowcharts (ගැලීම් සටහන්)

Flowchart එකකින් කරන්නේ programming logic එක රූපමය ආකාරයෙන් (visually) ඉදිරිපත් කිරීමයි. මේ සඳහා ජාත්‍යන්තරව පිළිගත් සංකේත භාවිතා කරනවා.

Scratch Programming

Scratch යනු මුලින්ම programming ඉගෙන ගන්නා අයට ඉතාමත් සුදුසු tool එකක්. මෙහි blocks drag කරමින් logic නිර්මාණය කළ හැකියි.

Motivation

“Programming ඉගෙන ගන්න ආරම්භ කරන්න – ලෝකයේ අනාගතය තීරණය වන්නේ තාක්ෂණය මතයි!”

ඔයාගේ O/L exam එකට සුබ පැතුම්! Flowcharts හොඳින් ඇඳලා පුහුණු වෙන්න.
