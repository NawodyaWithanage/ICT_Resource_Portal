Internet, Email සහ Web Browsers

Nawodya Withanage | December 17, 2025 | 12 මිනිත්තු

O/L ICT විෂයේ ඉතාම වැදගත් කොටසක් තමයි Internet, Email සහ Web Browsers ගැන ඉගෙන ගන්න එක. Practical එකේ email send කරන්න, browsing කරන්න එනවා. Theory එකේත් MCQ ගොඩක් එනවා.

Internet යනු කුමක්ද?

Internet යනු computers ගොඩක් interconnect වෙලා තියෙන global network එකක්. WWW (World Wide Web), Email, File Sharing වගේ services ලබාදෙනවා.

Web Browsers

Web pages view කරන්න භාවිතා වෙන software. Popular browsers: Google Chrome, Firefox, Edge, Safari.

  • URL type කරලා websites open කරන්න
  • Bookmarks, History, Tabs use කරන්න
  • Incognito mode privacy සඳහා

Email භාවිතය

Email යනු electronic mail. Gmail, Yahoo, Outlook වගේ services භාවිතා කරනවා.

  1. Compose > To: email address type කරන්න
  2. Subject සහ message ලියන්න
  3. Attachments add කරන්න (files, images)
  4. Send click කරන්න
  5. CC / BCC use කරලා copy send කරන්න

Internet Safety Tips

  • Strong passwords use කරන්න
  • Phishing emails වලට click නොකරන්න
  • HTTPS websites විතරක් use කරන්න
  • Antivirus install කරලා තියෙන්න

Motivations

“Internet එක safe විදිහට use කරන්න – knowledge ගන්න ලෝකය ඔයා ලඟ!”
“කවදාවත් ඉගෙන ගන්න නවත්තන්න එපා. දැනුම කවදාවත් නාස්ති වෙන්නේ නෑ.”
