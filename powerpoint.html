7. ඉදිරිපත් කිරීම් (MS PowerPoint)

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී Slides භාවිතයෙන් අදහස්, තොරතුරු සහ දත්ත දෘශ්‍යමය ලෙස ඉදිරිපත් කිරීම පිළිබඳව විස්තරාත්මකව ඉගෙන ගනිමු.

හැඳින්වීම

MS PowerPoint යනු Presentation Software එකකි. Slides භාවිතයෙන් අදහස්, තොරතුරු සහ දත්ත දෘශ්‍යමය ලෙස ඉදිරිපත් කිරීමට භාවිතා කරයි.

1. Presentation මූලික කොටස්

  • Presentation – PowerPoint file එක (.pptx)
  • Slide – Presentation හි එක් පිටුවක්
  • Slide Layout – Slide හි content සකස් වීමේ ආකෘතිය
  • Placeholder – Text / Image දාන්න තියෙන කොටස්

2. Slides නිර්මාණය කිරීම (Slide Creation)

2.1 නව Slide එකක් එකතු කිරීම

  • Home tab → New Slide
  • Layout එකක් තෝරන්න

2.2 Text ඇතුළත් කිරීම

  • Click placeholder → Text type කරන්න
  • Font size, color, alignment වෙනස් කළ හැක

2.3 Pictures / Shapes ඇතුළත් කිරීම

  • Insert tab → Pictures / Shapes
  • Images resize, move කළ හැක

3. Design සහ Themes

  • Design tab → Theme select කරන්න
  • Background color / image වෙනස් කළ හැක
  • Consistency (සමරූපතාව) වැදගත්ය

4. Transitions (සංක්‍රාන්ති)

Slide එකෙන් තවත් slide එකකට යන විදිහ Transition එකක් වේ.

  • Transitions tab → Transition type තෝරන්න
  • Duration / Sound set කළ හැක
  • Apply To All භාවිතයෙන් සියලු slides වලට දාන්න

5. Animations (ඇනිමේෂන්)

Slide එක ඇතුළත text / image move වීම Animation ලෙස හඳුන්වයි.

  • Animation tab → Object select කරන්න
  • Animation type (Entrance, Emphasis, Exit)
  • Animation Pane භාවිතයෙන් order control කරන්න

6. Slide Show ක්‍රියාත්මක කිරීම

  • Slide Show tab → From Beginning
  • F5 key භාවිතයෙන් presentation start කරන්න
  • Esc → Exit

7. Presentation Save & Print

  • File → Save / Save As (.pptx ලෙස save වේ)
  • File → Print → Slides / Handouts

8. වැදගත් උපදෙස් (Presentation Tips)

  • Slide එකකට අධික text දාන්න එපා
  • Font size පැහැදිලිව පෙනෙන ලෙස තෝරන්න
  • Animations & Transitions අධිකව භාවිතා නොකරන්න

වැඩිදුර අධ්‍යාපනය සඳහා

ඉදිරිපත් කිරීම් පිළිබඳ මෙම වීඩියෝ මාලාව නරඹන්න.

වීඩියෝව නරඹන්න

පුහුණු ප්‍රශ්න

  1. Slide Layout කියන්නේ මොකක්ද?
  2. Transition සහ Animation අතර වෙනස ලියන්න.
  3. Presentation start කරන්න භාවිතා කරන key එක කුමක්ද?
  4. Theme භාවිතා කිරීමේ ප්‍රයෝජනයක් ලියන්න.
