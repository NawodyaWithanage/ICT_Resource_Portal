9. මූලික වැඩසටහන්කරණ සංකල්ප

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී වැඩසටහන්කරණයේ පදනම වන ඇල්ගොරිතම, ප්‍රවාහ රූප සටහන්, සූඩෝකෝඩ් සහ මූලික පාලන ව්‍යුහයන් පිළිබඳව ඉගෙන ගනිමු.

හැඳින්වීම

ඇල්ගොරිතමයක් (Algorithm) යනු යම් කාර්යයක් ඉටු කිරීම සඳහා පියවරෙන් පියවර සකස් කරන ලද උපදෙස් මාලාවකි. වැඩසටහන්කරණයේ පදනම මෙයයි.

1. ඇල්ගොරිතමයක මූලික පියවර

ඕනෑම ඇල්ගොරිතමයක් ප්‍රධාන පියවර තුනකින් සමන්විත වේ:

  • ආදානය (Input): වැඩසටහනට දත්ත ලබා ගැනීම.
  • සැකසීම (Process): දත්ත මත ගණනය කිරීම් හෝ වෙනත් මෙහෙයුම් සිදු කිරීම.
  • ප්‍රතිදානය (Output): ප්‍රතිඵලය පෙන්වීම.

2. ප්‍රවාහ රූප සටහන් (Flowcharts)

ප්‍රවාහ රූප සටහනක් යනු ඇල්ගොරිතමයක පියවර සලකුණු (Symbols) මඟින් දෘශ්‍යමය ලෙස නිරූපණය කිරීමයි.

ප්‍රධාන ප්‍රවාහ රූප සටහන් සලකුණු

සලකුණ නම කාර්යය
ඕවල්ආරම්භය/අවසානය (Start/End)ඇල්ගොරිතමයක් ආරම්භ වන හෝ අවසන් වන ස්ථානය.
සමාන්තරාශ්‍රයආදානය/ප්‍රතිදානය (Input/Output)දත්ත ලබා ගැනීම හෝ ප්‍රතිඵල පෙන්වීම.
සෘජුකෝණාශ්‍රයක්‍රියාවලිය (Process)ගණනය කිරීම් හෝ දත්ත සැකසීම් සිදු කිරීම.
දියමන්තිතීරණය (Decision)කොන්දේසියක් පරීක්ෂා කිරීම (Yes/No).
ඊතලයප්‍රවාහ රේඛාව (Flowline)පියවර අතර දිශාව පෙන්වීම.

3. සූඩෝකෝඩ් (Pseudocode)

සූඩෝකෝඩ් යනු කිසිදු නිශ්චිත කේත භාෂාවක් භාවිතා නොකර, සාමාන්‍ය භාෂාවෙන් ඇල්ගොරිතමයක පියවර විස්තර කිරීමයි.

උදාහරණය: සංඛ්‍යා දෙකක එකතුව සෙවීම

START
    GET Number1
    GET Number2
    CALCULATE Sum = Number1 + Number2
    DISPLAY Sum
END

4. විචල්‍ය (Variables) සහ දත්ත වර්ග

විචල්‍යයක් යනු පරිගණකයේ මතකයේ දත්ත තාවකාලිකව ගබඩා කිරීම සඳහා වෙන් කර ඇති නමක් සහිත ස්ථානයකි.

4.1. දත්ත වර්ග (Data Types)

  • Integer (පූර්ණ සංඛ්‍යා): දශම රහිත සංඛ්‍යා (උදා: 10, -50).
  • Real/Float (දශම සංඛ්‍යා): දශම ලක්ෂ්‍ය අඩංගු සංඛ්‍යා (උදා: 3.14, 99.9).
  • String (අක්ෂර මාලා): අකුරු, වචන හෝ වාක්‍ය (උදා: "Hello", "Name").
  • Boolean (බූලියන්): සත්‍ය (TRUE) හෝ අසත්‍ය (FALSE) අගයන්.

5. කොන්දේසි (Conditions)

යම් කොන්දේසියක් සත්‍යද නැද්ද යන්න මත පදනම්ව වැඩසටහනේ ක්‍රියාත්මක විය යුතු මාර්ගය තීරණය කිරීමට මෙය භාවිතා කරයි.

5.1. IF...THEN...ELSE

IF Mark >= 50 THEN
    DISPLAY "Pass"
ELSE
    DISPLAY "Fail"
END IF

6. ලූප් (Loops / පුනරාවර්තන)

යම් කේත කොටසක් නැවත නැවතත් ක්‍රියාත්මක කිරීමට ලූප් භාවිතා කරයි.

6.1. For Loop

FOR Count FROM 1 TO 5 DO
    DISPLAY "Iteration" + Count
END FOR

6.2. While Loop

SET X TO 1
WHILE X < 10 DO
    SET X TO X * 2
END WHILE

පුහුණු ප්‍රශ්න

  1. ඇල්ගොරිතමයක් සකස් කිරීමේ ප්‍රධාන පියවර තුන කුමක්ද?
  2. ප්‍රවාහ රූප සටහනක 'දියමන්ති සලකුණ' භාවිතා කරන්නේ කුමන කාර්යය සඳහාද?
  3. "While" ලූප් එකක් භාවිතා කිරීමේදී ඇති ප්‍රයෝජනයක් ලියන්න.
  4. විචල්‍යයක දත්ත වර්ගයක් (Data Type) තීරණය කරන්නේ ඇයි?
