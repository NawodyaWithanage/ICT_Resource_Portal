Microsoft Word, Excel සහ PowerPoint භාවිතය

Nawodya Withanage | December 17, 2025 | 15 මිනිත්තු

O/L ICT practical exam එකේ ඉතාම වැදගත් කොටසක් තමයි Microsoft Office applications භාවිතය. මෙතනදී Word (mail merge), Excel (formulas, charts), PowerPoint (animations) ගැන step-by-step බලමු.

Microsoft Word - Mail Merge

Mail merge යනු එකම letter එකක් ගොඩක් අයට personalize කරලා send කරන විදිහයි.

  1. Mailings tab එකට යන්න.
  2. Start Mail Merge > Letters choose කරන්න.
  3. Select Recipients > Excel file එකක් choose කරන්න.
  4. Insert Merge Field > fields insert කරන්න.
  5. Finish & Merge ක්ලික් කරන්න.

Microsoft Excel - Formulas සහ Charts

Excel එකේ formulas භාවිතා කරලා calculations කරනවා. Charts data visually represent කරනවා.

ප්‍රධාන formulas: =SUM(A1:A10), =AVERAGE(B1:B10), =IF(C1>50, "Pass", "Fail").

Microsoft PowerPoint - Animations

  1. Object select කරන්න.
  2. Animations tab > Add Animation choose කරන්න.
  3. Animation Pane එකෙන් order සහ timing change කරන්න.
