12. බහුමාධ්‍ය සහ වෙබ් නිර්මාණ මූලික

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී බහුමාධ්‍ය මූලිකාංග (රූප, ශබ්ද, වීඩියෝ) සහ වෙබ් අඩවි වල පදනම වන HTML පිළිබඳව ඉගෙන ගනිමු.

හැඳින්වීම

බහුමාධ්‍ය (Multimedia) යනු තොරතුරු ඉදිරිපත් කිරීම සඳහා පෙළ (Text), රූප (Images), ශබ්ද (Audio), වීඩියෝ (Video) සහ සජීවීකරණ (Animation) වැනි විවිධ මාධ්‍ය සංයෝජනය කිරීමයි.

1. බහුමාධ්‍ය මූලිකාංග

1.1. රූප (Images)

පරිගණකයේ රූප ගබඩා කිරීමේ ප්‍රධාන ආකාර දෙකකි:

රූප වර්ගය විස්තරය උදාහරණ ගොනු ආකෘති
Bitmap / Raster Images තිත් (Pixels) සමූහයකින් සමන්විත වේ. විශාල කළ විට රූපයේ ගුණාත්මකභාවය අඩු වේ. JPEG, PNG, GIF, BMP
Vector Images ගණිතමය සමීකරණ මඟින් නිරූපණය වේ. ඕනෑම ප්‍රමාණයකට විශාල කළ විට ගුණාත්මකභාවය නොවෙනස්ව පවතී. SVG, AI, EPS

1.2. ශබ්ද (Audio)

  • Waveform / Digitized Audio: සැබෑ ශබ්ද තරංග නියැදි (Samples) ලෙස ලබාගෙන ගබඩා කිරීම. (උදා: WAV).
  • MIDI (Musical Instrument Digital Interface): ශබ්දය වෙනුවට, සංගීත උපදෙස් ගබඩා කරයි. ගොනු ප්‍රමාණය ඉතා කුඩාය.

1.3. වීඩියෝ (Video)

වීඩියෝ යනු අනුප්‍රාප්තික රූප රාමු (Sequence of Frames) වේගයෙන් ප්‍රදර්ශනය කිරීමයි.

  • Frame Rate (රාමු අනුපාතය): තත්පරයකට පෙන්වන රූප රාමු ගණන (FPS).
  • ජනප්‍රිය ගොනු ආකෘති: MP4, AVI, MOV.

2. වෙබ් නිර්මාණ මූලික: HTML හැඳින්වීම

HTML (HyperText Markup Language) යනු වෙබ් පිටු වල අන්තර්ගතය සහ ව්‍යුහය නිර්මාණය කිරීමට භාවිතා කරන සම්මත Markup භාෂාවයි.

2.1. HTML මූලික ව්‍යුහය

<!DOCTYPE html> <html lang="si"> <head> <meta charset="UTF-8"> <title>වෙබ් පිටුවේ මාතෘකාව</title> </head> <body> <h1>ප්‍රධාන මාතෘකාව</h1> <p>මෙය ඡේදයකි.</p> </body> </html>

2.2. බහුලව භාවිතා වන HTML ටැග් (Tags)

HTML ටැගය කාර්යය උදාහරණය
<h1> to <h6> මාතෘකා නිර්මාණය කිරීමට. <h1>මාතෘකාව</h1>
<p> ඡේදයක් නිර්මාණය කිරීමට. <p>ඡේදයකි.</p>
<a> සබැඳියක් (Hyperlink) නිර්මාණය කිරීමට. <a href="#">Link</a>
<img> රූපයක් ඇතුළත් කිරීමට. <img src="img.jpg">

2.3. Attributes (ගුණාංග)

Attributes යනු ටැග් එකකට අමතර තොරතුරු සැපයීමට භාවිතා කරන යෙදුම් වේ. (උදා: <img> ටැගයේ ඇති src, alt).

වැඩිදුර අධ්‍යාපනය සඳහා

බහුමාධ්‍ය සහ HTML පිළිබඳ මෙම වීඩියෝව නරඹන්න.

වීඩියෝව නරඹන්න

පුහුණු ප්‍රශ්න

  1. Bitmap (Raster) සහ Vector රූප ආකෘති අතර ඇති ප්‍රධාන වෙනස කුමක්ද?
  2. ශබ්ද ගොනු වර්ග දෙකක් නම් කරන්න.
  3. HTML ලේඛනයක <head> ටැගය තුළ අඩංගු විය යුත්තේ කුමක්ද?
  4. HTML ගුණාංගයක (Attribute) කාර්යය කුමක්ද? උදාහරණයක් දෙන්න.
නිබන්ධන මෙනුවට ආපසු
