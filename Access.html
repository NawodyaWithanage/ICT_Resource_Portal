8. දත්ත සමුදායන්

MICROSOFT ACCESS GUIDE

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී MS Access භාවිතයෙන් දත්ත සමුදායක් (Database) නිර්මාණය කර, කළමනාකරණය කරන ආකාරය පිළිබඳව විස්තරාත්මකව ඉගෙන ගනිමු.

හැඳින්වීම

දත්ත සමුදායක් යනු ක්‍රමවත් ලෙස ගබඩා කරන ලද එකිනෙකට සම්බන්ධිත දත්ත සමූහයකි.

MS Access යනු එවැනි දත්ත සමුදායන් නිර්මාණය කිරීමට හා කළමනාකරණය කිරීමට භාවිතා කරන මෘදුකාංගයකි.

දත්ත සමුදායක ප්‍රධාන කොටස් (Objects)

MS Access දත්ත සමුදායක් ප්‍රධාන කොටස් හතරකින් (4) සමන්විත වේ.


  • වගු (Tables): දත්ත සැබවින්ම ගබඩා වන ස්ථානය.
  • විමර්ශන (Queries): වගු වලින් දත්ත තෝරා ගැනීමට, පෙරීමට, ගණනය කිරීමට සහ වෙනස් කිරීමට භාවිතා කරයි.
  • ආකෘති (Forms): දත්ත ඇතුළු කිරීමට, සංස්කරණය කිරීමට සහ බැලීමට පහසු පරිශීලක අතුරුමුහුණතක් නිර්මාණය කරයි.
  • වාර්තා (Reports): මුද්‍රණය සඳහා සූදානම් කළ හැකි ආකාරයෙන් දත්ත විශ්ලේෂණය කිරීමට හෝ සාරාංශගත කිරීමට භාවිතා කරයි.

වගු නිර්මාණය (Tables)

වගුවක් නිර්මාණය කිරීමේදී Field Name සහ Data Type තෝරාගත යුතුය.වගුවක් තීරු (ක්ෂේත්‍ර/Fields) සහ පේළි (වාර්තා/Records) වලින් සමන්විත වේ.


වගුවක මූලිකාංග

  • ක්ෂේත්‍ර (Fields): වගුවක තීරුවකි. (උදා: නම, ලිපිනය).
  • වාර්තා (Records): වගුවක පේළියකි. (එක් පුද්ගලයෙකු පිළිබඳ සියලු තොරතුරු).
  • ප්‍රාථමික යතුර (Primary Key): එක් වාර්තාවක් තනිව හඳුනා ගැනීමට භාවිතා කරන ක්ෂේත්‍රයකි. (වාර්තා අනුපිටපත් වීම වළක්වයි).

වගුවක් නිර්මාණය කිරීම

  1. Create tab වෙත යන්න.
  2. Table Design තෝරන්න.
  3. Field Name සහ Data Type තීරුවල ක්ෂේත්‍ර ඇතුළත් කරන්න.
    (ප්‍රධාන වශයෙන් භාවිතා වන දත්ත වර්ග: Short Text, Number, Date/Time.)
  5. අවශ්‍ය ක්ෂේත්‍රය තෝරා Design tab එකේ ඇති Primary Key බොත්තම ඔබා ප්‍රාථමික යතුරක් සකසන්න.
  6. වගුව සුරකින්න (Save) සහ එයට නමක් දෙන්න.
  7. Datasheet View වෙත මාරු වී දත්ත ඇතුළු කරන්න.

විමර්ශන නිර්මාණය (Queries)

වගු වලින් අවශ්‍ය දත්ත පමණක් තෝරා ගැනීමට, පෙරීමට, හෝ ගණනය කිරීමට විමර්ශන භාවිතා කරයි.


විමර්ශනයක් නිර්මාණය කිරීම

  • Create tab වෙත යන්න.
  • Query Design තෝරන්න.
  • විමර්ශනයට අවශ්‍ය වගු (Tables) තෝරා Add කරන්න.
  • අවශ්‍ය ක්ෂේත්‍ර (Fields) පහළ ග්‍රිඩයට ඇද දමන්න (Drag) හෝ Double Click කරන්න.
  • Criteria පේළියට ගොස් කොන්දේසි (Criteria) ඇතුළු කරන්න.
    (උදා: Marks ක්ෂේත්‍රය යටතේ >50 ලෙස ඇතුළත් කිරීම.)

ආකෘති නිර්මාණය (Forms)

දත්ත ඇතුළු කිරීමේ ක්‍රියාවලිය පහසු කරවීම සඳහා ආකෘති භාවිතා වේ. මේවා තොරතුරු යෙදවීම/දැක්වීම සඳහා අතුරුමුහුණතක් (User Interface) සපයයි.


  • ආකෘතියට පදනම් කර ගන්නා වගුව (Table) තෝරන්න.
  • Create tab වෙත යන්න.
  • Form බොත්තම ක්ලික් කරන්න. (Access විසින් ස්වයංක්‍රීයව ආකෘතියක් නිර්මාණය කරනු ඇත.)
  • Design View හෝ Layout View භාවිතයෙන් ආකෘතිය සංස්කරණය කළ හැක.

වාර්තා නිර්මාණය (Reports)

දත්ත සාරාංශගත කර, සංවිධානය කර, මුද්‍රණය සඳහා සුදුසු ආකාරයෙන් ඉදිරිපත් කිරීමට වාර්තා භාවිතා කරයි.


  • වාර්තාවට පදනම් කර ගන්නා වගුව හෝ විමර්ශනය තෝරන්න.
  • Create tab වෙත යන්න.
  • Report බොත්තම ක්ලික් කරන්න. (Access විසින් ස්වයංක්‍රීයව වාර්තාවක් නිර්මාණය කරනු ඇත.)
  • Print Preview මඟින් මුද්‍රණයේදී වාර්තාව දිස්වන ආකාරය දැක ගත හැක.

වැඩිදුර අධ්‍යාපනය සඳහා

පියවරෙන් පියවර MS Access භාවිතා කරන ආකාරය මෙතැනින් නරඹන්න.

වීඩියෝව නරඹන්න

පුහුණු අභ්‍යාස

  1. දත්ත සමුදායක ප්‍රාථමික යතුරක් (Primary Key) භාවිතා කිරීමේ ප්‍රධාන අරමුණ කුමක්ද?
  2. වගුවක ඇති වාර්තාවක් (Record) යනු කුමක්ද?
  3. විමර්ශන (Queries) භාවිතා කරන්නේ කුමන කාර්යයන් සඳහාද?
  4. දත්ත සමුදායක ආකෘති (Forms) සහ වාර්තා (Reports) අතර ඇති මූලික වෙනස පැහැදිලි කරන්න.
