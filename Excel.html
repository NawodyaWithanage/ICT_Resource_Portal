6. විස්තීර්ණ පත්‍ර (MS Excel)

සාදරයෙන් පිළිගනිමු මෙම නිබන්ධනයට!
මෙහිදී අපි ගණනය කිරීම්, දත්ත සංවිධානය කිරීම, ලකුණු පත්‍ර සෑදීම සහ Charts භාවිතයෙන් දත්ත දෘශ්‍යමය ලෙස පෙන්වීම පිළිබඳව විස්තරාත්මකව ඉගෙන ගනිමු.

හැඳින්වීම

MS Excel යනු Spreadsheet Software එකකි. ගණනය කිරීම්, දත්ත සංවිධානය කිරීම, ලකුණු පත්‍ර සෑදීම සහ Charts භාවිතයෙන් දත්ත දෘශ්‍යමය ලෙස පෙන්වීමට මෙය භාවිතා කරයි.

1. MS Excel හි මූලික කොටස්

  • Workbook – Excel file එක (Book1.xlsx)
  • Worksheet – Workbook එක ඇතුළත ඇති sheet
  • Row – 1,2,3 ලෙස පේළි
  • Column – A,B,C ලෙස තීරු
  • Cell – Row + Column එකතුව (A1, B2)

2. දත්ත ඇතුළත් කිරීම (Entering Data)

  • Text – නම, විෂය
  • Number – ලකුණු, මුදල්
  • Date – 2025/01/01

3. Formulas (සූත්‍ර)

Excel හි සෑම Formula එකක්ම = ලකුණෙන් ආරම්භ වේ.

සූත්‍රයභාවිතය
=A1+B1එකතු කිරීම
=A1-B1අඩු කිරීම
=A1*B1ගුණ කිරීම
=A1/B1බෙදීම

4. Functions (කාර්යයන්)

Functions කියන්නේ Excel තුළ තිබෙන සූදානම් සූත්‍ර වේ.

  • SUM – =SUM(A1:A5)
  • AVERAGE – =AVERAGE(A1:A5)
  • MAX – =MAX(A1:A5)
  • MIN – =MIN(A1:A5)
  • COUNT – =COUNT(A1:A5)

5. Practical Example – Marks Sheet

නමගණිතයවිද්‍යාවඉංග්‍රීසිමුළු ලකුණුසාමාන්‍යය
Amal657075=SUM(B2:D2)=AVERAGE(B2:D2)
Nimali807885=SUM(B3:D3)=AVERAGE(B3:D3)

Fill Handle භාවිතයෙන් formulas පහළට drag කර භාවිතා කළ හැක.

6. Charts

Charts භාවිතයෙන් දත්ත දෘශ්‍යමය ලෙස පෙන්වයි.

  • Column Chart
  • Bar Chart
  • Pie Chart
  • Line Chart

Chart එකක් සෑදීමේ පියවර

  1. Data select කරන්න
  2. Insert tab → Charts
  3. Chart Type එක තෝරන්න
  4. Chart Title add කරන්න

වැඩිදුර අධ්‍යාපනය සඳහා

විස්තීර්ණ පත්‍ර පිළිබඳ මෙම වීඩියෝව නරඹන්න.

වීඩියෝව නරඹන්න

පුහුණු ප්‍රශ්න

  1. A1 සිට A10 දක්වා එකතුව සඳහා Formula එක ලියන්න.
  2. වැඩිම අගය සොයන Function එක කුමක්ද?
  3. Chart එකක් සෑදීමේ පියවර 3ක් ලියන්න.
  4. Workbook සහ Worksheet අතර වෙනස ලියන්න.
